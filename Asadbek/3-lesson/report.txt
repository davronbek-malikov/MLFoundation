Machine learning (ML) loyiha qilish ketma ketliginig "Data toplash" bosqichi: 
- Sorovnoma
- Web saytlar orqali malumotlar yigish
- Github yoki Kaggle orqali dataset larni olish
- Mijoz

Data preprocessing bosqichida qiladigan ishlarimizdan biri.
Datasetimizni kozdan kechirib chiqish yani keraksiz ustun bormi yoki yoq. Bizni modelimiz ishlashi uchun ustunning aloqasi bormi yoki yoq.
Masalan bizda quyidagi ustunlarimiz mavjuda bolsa => userId, age, address, salary Bu holatda userning ID bizga keraksiz hisoblanadi.

Datasetimizda tushurib qoldirilgan qiymatlarni toldirish.
Bularning 5 xil usuli mavjud.

- MEAN => orta arifmetik qiymat bilan toldiriladi (bu usul faqat raqamli ustunlar uchun ishlaydi)
- MODE => eng kop takrorlangan qiymatlar bilan toldiriladi
- MEDIAN => ortasidagi qiymat bilan toldiriladi.
- FIXED => biz xoxlagan qiymat bilan toldiriladi (yani biz bu qiymatni oldindan aniq bilar edik)
- Tashlab yuborish bu bizda oz navbatida 2 ga bolinadi => ustun boyicha hamda qator boyicha tashlab yuborish
  - Ustun boyicha tashlab yuborishda biz quyidagi holatga etibor berishimiz kerak. 
  Ustunning 50% dan kop qiymati tushib qolgan bolsa biz tashlab yuborishimiz kerak chunki 50% dan kopini ozimiz taxminiy toldirsa bizning modelimiz ishlash aniqligi pasayadi.
